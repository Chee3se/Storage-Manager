## How to install

### 1. Clone the repository
### 2. install composer dependencies.
```bash
composer install
```
### 3. install node dependencies.
```bash
npm install
```
### 4. Copy the .env.example file to .env
```bash
cp .env.example .env
```
### 5. Migrate the database
```bash
php artisan migrate
```
### 6. Run the node development server
```bash
npm run dev
```
### 7. Run the server
```bash
php artisan serve
```
